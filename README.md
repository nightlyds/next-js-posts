# Posts :speech_balloon:

That`s testing project for oppotunity watch posts & adding posts.

## Installation

1. Use the package manager [yarn](https://yarnpkg.com/) to install packages.

2. Just type in terminal:

```bash
yarn install
```

and after all packages will downloaded:

```bash
yarn dev
```

3. Enjoy!


## Commands

Start development:

```bash
yarn dev
```

Typescript checking:

```bash
yarn tsc
```

EsLint and Prettier:

```bash
yarn lint
yarn pretty
yarn format
```

Start project:

```bash
yarn start
```

Build project:

```bash
yarn build
yarn build:server,
yarn build:next,
```

Test project:

```bash
yarn test
yarn test:coverage
yarn test:watch
```
